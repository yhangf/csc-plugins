### 2.2.2. **系统架构图** ⭐必填

<span style="color:orange">**[必填图示] 使用 Mermaid 或 PlantUML 绘制，禁止只用 ASCII 图代替。图后必须附架构说明文字（AI文字描述）。**</span>

**图示（Mermaid示例，按实际替换）：**

```mermaid
graph TD
    User([用户/外部系统])

    subgraph System["本系统"]
        SubA["子系统A\n(描述职责)"]
        SubB["子系统B\n(描述职责)"]
        SubC["子系统C\n(描述职责)"]
    end

    ExternalDB[(外部数据库)]
    ExternalSvc["外部服务"]

    User -->|"请求描述"| SubA
    SubA -->|"调用描述"| SubB
    SubB -->|"读写"| ExternalDB
    SubA -->|"调用描述"| SubC
    SubC -->|"调用"| ExternalSvc
```

| *子系统B* | *内部子系统* | *负责xxx* | *被子系统A调用，读写外部数据库* |
