## 高密度摘要

**文档类型**：TR1 用户需求规格说明书（具体产品版本填写用）

**全局强约束（填写时必须遵守）**：

- **必须**：客户验证 ≥ 3 个，结论有据可查（对应 QS-001）
- **必须**：协作方需求已与协作方规划主管达成书面共识（对应 QS-002）
- **必须**：P0/P1 严重问题已纳入需求池（对应 QS-003）
- **必须**：DFX 需求基线全量落地，不满足项有 Story 跟踪（对应 QS-004）
- **必须**：单个需求包含用户原始诉求、现状旅程、痛点、未来旅程、场景描述（对应 QS-005）
- **必须**：海外产品完成海外需求基线自检（对应 QS-006）
- **必须**：完成道德合规 Checklist TR1 阶段 4 类检视（对应 QS-007）
- **必须**（XAAS 类）：包含客户成功基线需求并完成自检（对应 QS-008）
- **禁止**：使用"合理 / 适当 / 较好 / 尽量 / 性能好 / 适当考虑"等模糊词

**8 条 QS 必过项速查**：

| QS 编号 | 检查项 | 裁剪原则 | 对应章节 |
|---|---|---|---|
| QS-001 | 完成客户验证（≥3 个） | 不可裁剪 | §客户验证结论 |
| QS-002 | 完成协作方需求共识 | 无协作需求可裁剪 | §协作方需求共识 |
| QS-003 | 严重产品问题纳入需求池 | 不可裁剪 | §质量需求 |
| QS-004 | 明确 DFX 需求基线 | 不可裁剪 | §2.2 DFX 需求基线 |
| QS-005 | 单个需求包含基本内容 | 产线可自定义模板 | §1.1.4 特性详细需求规格 |
| QS-006 | 完成海外需求基线自检 | 不涉及出海可裁剪 | §国际化支持 |
| QS-007 | 完成道德合规 TR1 检视 | 不可裁剪 | §2.2.4 基础属性 — 合规性 |
| QS-008 | 包含客户成功相关需求 | 非 XAAS 可裁剪 | §XAAS 客户成功 |
