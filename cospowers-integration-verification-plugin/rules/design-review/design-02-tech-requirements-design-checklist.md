# 模块 M2：技术需求设计

## 模块概述

**审查目的**：确保设计文档的技术需求分析充分、推导过程合理、方案选型有依据，为后续概要设计/总体设计提供坚实的基础。

**适用范围**：总体设计文档、技术需求设计文档。

**说明**：本模块共 7 个检查项。

---

## 01. 技术需求设计

### 1.1 架构需求分析

**检查内容**：
1. 涉及架构变动的项目需填写此章节，包含但不限于以下类型：
   - 新产品、新架构、重大重构版本
   - 涉及跨产线的架构融合项目
2. 基于场景分析→问题识别→技术思路→方案选型的梳理推导过程，确保需求场景无遗漏，推导过程充分、合理
3. 架构目标具有前瞻性，充分考虑了产品长期演进的潜在需求、解决方案层面的潜在需求、中台需求，以及架构在全球业内领先性演进的需求。内容已经通过架构师或 TL 的认可
4. 不涉及的需明确说明不涉及

**评判标准**：
- 通过：架构需求分析充分完整，推导路径清晰，有一定前瞻性
- 不通过：推导过程跳跃、场景遗漏、未考虑长期演进需求，或涉及架构变动但本章节为空
- 不适用：不涉及架构变动的项目，且已明确说明

### 1.2 特性需求分析

**检查内容**：按附件文档要求填写，基于场景分析→问题识别→技术思路→方案选型的梳理推导过程，确保需求场景无遗漏，推导过程充分、合理。

**评判标准**：
- 通过：特性需求分析有完整的场景→问题→思路→选型推导链
- 不通过：需求场景有明显遗漏，或推导过程跳跃、缺乏逻辑
- 不适用：按实际情况判断

**通过示例**：
> "用户场景：分支员工在弱网环境下上传大文件（>100MB），痛点：传输中断需重传。技术思路：分片断点续传。方案选型：采用 HTTP Range 分片 + Redis 记录分片状态，相比 WebSocket 方案更适合弱网环境。"

**不通过示例**：
> "需要支持文件上传，选择 HTTP 上传方式实现"
> （缺少场景分析和方案对比推导）

### 1.3 产品预期目标

**检查内容**：用户视角、用户可感知的产品预期目标，需要考虑当前版本以及未来演进的目标，起到牵引作用。

**评判标准**：
- 通过：有明确的、可量化的、用户视角的产品目标，并考虑了版本演进
- 不通过：目标不明确、不可量化、纯技术视角，或未区分当前版本和未来演进

**通过示例**：
> "V4.0 目标：分支与总部间文件传输成功率从 85% 提升到 99%，传输延迟降低 30%；V4.1 规划：支持跨国加速"

**不通过示例**：
> "目标：提高传输性能"（缺乏量化和用户视角）

### 1.4 技术预研和中台复用

**检查内容**：按要求填写"产品技术风险消除 checklist"附件，技术预研项目有明确的结论。若是带风险通过，还需要在方案选型中体现具体的风险细节。有中台复用的结论，后续要有方案选型过程。

**评判标准**：
- 通过：技术预研有结论，风险项在方案选型中有体现，中台复用有明确结论和选型过程
- 不通过：技术预研结论模糊，带风险通过但未在方案选型中体现风险，或中台复用结论缺失
- 不适用：无技术风险项且无中台复用场景，已明确说明

### 1.5 方案选型

**检查内容**：
1. 要包含技术预研、中台复用的选型
2. 方案选型明确说明每个方案的优缺点，要基于业务识别出来的关键技术维度进行评估，包括但不限于功能、性能、可测试性、可扩展性。最终采用的是哪个方案，选择该方案的具体原因。选择该方案必须得到架构师确认（上 TMG 评审的，需通过 TMG 认可）

**评判标准**：
- 通过：方案对比维度完整，优缺点分析客观，选型理由充分，有架构师确认记录
- 不通过：只有一个方案无对比、优缺点分析片面、选型理由不充分、缺少架构师确认

## 02. 设计目标

### 2.1 设计目标细化

**检查内容**：
1. 基于"产品预期目标"进一步细化各设计指标，包括用户可感知和不可感知的设计目标
2. 每个维度的设计目标设定要有依据支撑，"总设设计目标分析"附件需填写完整
3. 不涉及的明确说明不涉及，内容已经通过架构师或 TL 的认可

**评判标准**：
- 通过：设计目标覆盖用户可感知和不可感知维度，有依据支撑，有架构师/TL 认可
- 不通过：设计目标维度单一（如只有性能指标），缺少依据说明，或缺少架构师/TL 认可
- 不适用：已明确说明不涉及

### 2.2 中台技术要求

**检查内容**：如存在可复用中台，优先考虑使用中台提升研发效率。需列出关键的中台技术要求，并且与中台负责人确认。

**评判标准**：
- 通过：有中台复用需求时，列出了关键技术要求并与中台负责人确认
- 不通过：有中台复用需求但未列出技术要求或未经中台负责人确认
- 不适用：无可复用中台，已明确说明
